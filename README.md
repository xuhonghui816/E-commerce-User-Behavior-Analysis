# 电商用户行为分析与购买预测



## 项目介绍:

这是一个完整的电商数据分析项目，基于真实数据集，包含数据清洗、探索分析、客户分群、机器学习建模和业务洞察。



##### 快速开始

1\. 安装依赖：`pip install -r requirements.txt`

2\.打开 `电商数据分析.ipynb`

3\. 运行代码块



##### 项目流程

1\. 数据加载与清洗

2\.探索性数据分析

3\.RFM客户分群

4\.购买预测建模

5\.业务建议



##### 技术栈

Python, Pandas, Scikit-learn,RandomForest，XGBoost, Matplotlib



📊 项目概述
这个项目分析了英国一家在线零售商店的真实销售数据，时间跨度为2010年12月至2011年12月。通过数据清洗和分析，揭示了销售规律、客户行为等重要商业洞察。

🔍 核心指标解析
1. 数据基本情况
原始数据：541,909条交易记录

清洗后数据：397,884条有效记录

数据字段：包括商品信息、销售数量、价格、客户信息等

通俗理解：相当于分析了近40万条有效的购物记录！

2. 数据清洗过程
项目对原始数据进行了重要处理：

删除无效客户：删除了13.5万条没有客户ID的记录

过滤异常数据：删除了数量或价格为负数的异常记录

创建新指标：计算了销售额、分析时间维度等

为什么重要：就像打扫房间一样，清理掉脏数据才能得到准确的分析结果。

📈 关键发现与分析
1. 销售金额分析

基本统计信息：
- 平均每笔交易：£22.40
- 最大单笔交易：£168,469（可能是大宗采购）
- 75%的交易在£19.80以下
商业意义：大部分交易金额不大，但存在少量大额订单，说明既有日常零售也有批发业务。

2. 地域销售分布
英国占据绝对主导地位，其次是荷兰、爱尔兰、德国等欧洲国家。

业务启示：

英国是核心市场，应重点维护

其他欧洲国家有增长潜力

可以考虑针对性营销策略

3. 时间趋势分析
📅 月度销售趋势
明显季节性：2011年11月出现销售高峰（圣诞购物季）

增长态势：整体销售额呈上升趋势

年初低谷：2011年1月销售较低（节日后效应）

商业价值：为库存管理、促销安排提供时间参考。

📆 每周销售模式
text
销售额排名：周四 > 周二 > 周三 > 周五 > 周一 > 周六 > 周日
洞察：工作日销售明显好于周末，周四是最佳销售日。

⏰ 每日销售时段
高峰时段：中午12点至下午3点

低谷时段：早上7点前和晚上8点后

平稳期：上午10点到下午6点

应用价值：指导客服排班、促销活动 timing。

4. 客户价值分析

客户概况：
- 总客户数：4,372名
- 平均客户价值：£1,335.99
- 平均订单数：5.4次
重要发现：

复购率不错：平均每个客户下单5次以上

客户忠诚度：存在稳定的客户群体
