电商用户行为分析与购买预测

项目流程
1.数据加载与清洗
2.探索性数据分析
3.RFM客户分群
4.购买预测建模
5.业务建议

技术栈
Python, Pandas, Scikit-learn,RandomForest，XGBoost, Matplotlib.......

项目概述
这个项目分析了英国一家在线零售商店的真实销售数据，时间跨度为2010年12月至2011年12月
通过数据清洗和分析，揭示了销售规律、客户行为等重要商业洞察

一、核心指标解析
1. 数据基本情况
- 原始数据：541,909条交易记录
- 清洗后数据：397,884条有效记录
- 数据字段：包括商品信息、销售数量、价格、客户信息等
- 通俗理解：分析了近40万条有效的购物记录
2. 数据清洗过程
项目对原始数据进行了重要处理：
- 删除无效客户：删除了13.5万条没有客户ID的记录
- 过滤异常数据：删除了数量或价格为负数的异常记录
- 创建新指标：计算了销售额、分析时间维度等

二、关键发现与分析
1. 销售金额分析
基本统计信息：
- 平均每笔交易：£22.40
- 最大单笔交易：£168,469（可能是大工厂或公司采购）
- 75%的交易在£19.80以下
商业意义：大部分交易金额不大，但存在少量大额订单，说明既有日常零售也存在一些大批发业务
2. 地域销售分布
业务启示
- 英国占据绝对主导地位，其次是荷兰、爱尔兰、德国等欧洲国家，英国是核心市场，应重点维护
- 其他欧洲国家有增长潜力
- 可以考虑针对性营销策略
3. 时间趋势分析
月度销售趋势
- 明显季节性：2011年11月出现销售高峰（圣诞购物季）
- 增长态势：整体销售额呈上升趋势
- 年初低谷：2011年1月销售较低（节日后效应）
- 商业价值：为库存管理、促销安排提供时间参考
每周销售模式
- 销售额排名：周四 > 周二 > 周三 > 周五 > 周一 > 周六 > 周日
- 洞察：工作日销售明显好于周末，周四是最佳销售日（跟我国预想的不太一样）
每日销售时段
- 高峰时段：中午12点至下午3点
- 低谷时段：早上7点前和晚上8点后
- 平稳期：上午10点到下午6点
- 应用价值：指导客服进行排班、搞点促销活动
4. 客户价值分析
客户概况：
- 总客户数：4,372名
- 平均客户价值：£1,335.99
- 平均订单数：5.4次
重要发现：
- 客户价值较高：平均每个客户贡献超过1300英镑
- 复购率不错：平均每个客户下单5次以上
- 客户忠诚度：存在稳定的客户群体

三、商业建议
1. 营销策略优化
- 重点时段投放：在周四和中午时段加大广告投入
- 季节性准备：提前为11月销售高峰备货
- 欧洲市场拓展：在英国之外寻找增长机会
2. 运营效率提升
- 客服排班：在销售高峰时段安排更多客服人员
- 库存管理：根据销售趋势预测库存需求
3. 客户关系管理
- 高价值客户维护：重点服务贡献大的客户
- 提升复购率：通过会员计划增加客户粘性

四、建模预测：数据驱动的智能决策
1.客户购买预测模型
预测效果验证：
- 基于4,070名客户的历史数据，模型成功预测未来30天的购买行为：
- 预测准确率：71.89%（AUC得分）
- 高意向客户：1,380人（占总客户33.9%）
- 模型比较：Random Forest表现最佳，优于Logistic Regression和XGBoost
精准营销实施：
- 目标客户群：从4,070人精准缩小至1,380人
- 营销效率：提升2.8倍
- 资源节省：减少66%的营销覆盖范围
- 预期效果：在保持相同转化量的前提下大幅降低营销成本
2.销售预测模型
预测精度分析：
基于历史销售数据的时间序列预测：
- 解释能力：模型能解释72.98%的销售额变化（R² = 0.7298）
- 误差范围：平均预测偏差£6,860，最大偏差£13,748
- 实用价值：为日常运营提供可靠的销售预期
基于预测的优化：
- 缺货预防：提前识别销售高峰，避免库存不足
- 资金效率：减少过度库存造成的资金占用
- 采购规划：为采购决策提供数据支持
3.RFM客户价值分群
将客户分为5个群体，客户分布情况：
- 冠军客户：高活跃、高价值客户
- 忠实客户：稳定复购的优质客户
- 潜力客户：有成长空间的中等客户
- 一般客户：价值普通的常规客户
- 流失客户：长期未购买客户
客户管理策略：
- 重点维护：冠军客户和忠实客户（高价值群体）
- 培育开发：潜力客户（增长机会点）
- 成本优化：一般客户和流失客户（减少资源投入）
- 精准营销：针对不同群体制定差异化策略
4.模型综合业务价值
成本效益分析：
- 营销成本优化：通过精准定位减少无效营销支出
- 库存成本节约：基于预测的智能库存管理
- 服务效率提升：数据驱动的资源分配
- 客户价值最大化：重点维护高价值客户群体
战略价值：
- 决策支持：为营销、库存、客服等决策提供数据依据
- 风险控制：提前预见销售波动和客户流失风险
- 效率提升：全业务流程的数据驱动优化
竞争优势：建立在数据洞察基础上的差异化竞争力

五、项目总结
核心成果
- 客户洞察：通过RFM分析深入理解客户价值分布
- 行为预测：准确预测客户购买意向和销售趋势
- 决策优化：为营销、库存、服务等关键业务提供数据支持
- 效率提升：通过精准定位大幅提升资源使用效率
业务影响
- 短期价值：立即可实施的精准营销和库存优化方案
- 长期价值：建立数据驱动的决策体系和竞争优势
